To Access the smart system, start by connecting to the MQTT server using cmd by typing in mosquitto -v -p 1889

Then open node-red by typing in node-red in cmd, importing flows.json and deploying

On whatever coding IDE, load an new environment, download all the modules required in requirements.txt and open the file Project_UAS and type the command py manage.py runserver 0.0.0.0:8000

once the website starts running, type localhost:8000 on your browser.
